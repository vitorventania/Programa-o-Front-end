Questão 01

Introdução:

O HTML (HyperText Markup Language) é a linguagem fundamental para a construção de páginas web, sendo responsável por estruturar e organizar o conteúdo exibido no navegador. Por meio de tags, é possível definir elementos como títulos, parágrafos, links, imagens e também organizar semanticamente a página com estruturas como cabeçalho, navegação e seções de conteúdo. Considerando os conceitos apresentados em aula sobre estrutura básica de uma página HTML e uso de tags de organização de conteúdo, desenvolva a atividade a seguir.
Assim como os recursos do HTML, são a base estrutural da WEB moderna, como vimos com o distinto senhor Tim Berners Lee, o código HTML deste exercicio é base solida para as proximas duas atividades desta prova.

Comando:

Crie uma página HTML completa contendo a estrutura básica do documento (<html>, <head>, <title> e <body>).
Dentro da tag <body>, utilize as tags <div> | <h(1 a 6)> | <ul> e <li> | <details> de estruturação de conteúdo, para organizar as informações do Descritivo de Apoio abaixo como uma página da WEB.

A página deve conter obrigatoriamente:
  • As tags solicitadas;
  • A divisão dos itens (1,2 e 3) do descritivo de apoio em distintos itens dentro da página através de <div> ;
  • Os fatos sobre os grupos, devem estar contidos ao final de cada grupo em um item Saiba Mais da tag <details> ;
  • Uma organização clara e estruturada do conteúdo utilizando as tags solicitadas.
É permitida a utilização de demais tag's e recursos do HTML, porém é criterio avaliativo os itens acima mencionados.
Pode ser interessante para o desenvolvimento da página se cada um dos itens possuir um Id definido.

Questão 02

Introdução:

O CSS (Cascading Style Sheets) permite a separação entre o conteúdo (HTML) e a apresentação visual. A utilização de folhas de estilo externas é a prática recomendada na indústria, pois facilita a manutenção do código e a padronização visual em múltiplas páginas de um mesmo projeto.
Lembrando que a estrutura do CSS possui 3 itens:
  • Seletor: Identifica o elemento HTML.
  • Propriedade: O atributo que será alterado (ex: color).
  • Valor: A configuração aplicada (ex: blue).
A configuração utlizada no exercicio (interno, externo, inline) é indifernte, desde que aplicado o CSS

Comando:

Crie uma "folha de estilos"(CSS), para estilizar a página desenvolvida no item anterior.
Que aplique minimamente as Propriedades do CSS:
border-radius | padding | background-color e,ou background| | display do tipo felx | aling-itens e justify-content para alinhamento | color | border).
Com valores irrestritos, utilize da forma que preferir o CSS para melhorar o aspecto visual das informações da página.

O estilo da página deve conter obrigatoriamente:
  • A estilização de classe do CSS para ser aplicada as a cada item da página .
  • O Cabeçalho da pagina centralizado
  • Uma cor de fundo suave aplicada ao background.
  • Que o destaque da pagina sejam os "cartões de itens" do descritivo
  • Itens da pagina minimamente espaçados entre si e alinhados
